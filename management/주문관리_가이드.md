# 📋 AI Studio 주문 관리 시스템

> 고객 주문을 효율적으로 관리하기 위한 템플릿입니다.
> Notion이나 Excel에서 사용할 수 있습니다.

---

## 📊 주문 관리 데이터베이스 구조

### 필수 필드

| 필드명 | 타입 | 설명 | 예시 |
|--------|------|------|------|
| 주문번호 | 텍스트 | 고유 식별자 | AS-2025-0001 |
| 주문일 | 날짜 | 주문 접수일 | 2025-11-30 |
| 고객명 | 텍스트 | 고객 이름/업체명 | 김철수 / ABC마켓 |
| 연락처 | 텍스트 | 이메일/전화번호 | example@email.com |
| 서비스 유형 | 선택 | 서비스 종류 | 제품이미지/썸네일/영상/브랜딩 |
| 패키지 | 선택 | 선택한 패키지 | Basic/Standard/Premium |
| 상태 | 선택 | 진행 상태 | 대기/진행중/검토중/완료 |
| 금액 | 숫자 | 결제 금액 | 100,000 |
| 납품예정일 | 날짜 | 예상 납품일 | 2025-12-02 |
| 메모 | 텍스트 | 추가 메모 | 급행 요청, 수정 1회 완료 |

### 상태 값 정의

| 상태 | 설명 | 색상 (Notion) |
|------|------|--------------|
| 💬 문의 | 견적 문의 단계 | 회색 |
| 💰 결제대기 | 견적 발송, 결제 대기 | 노랑 |
| 🎨 진행중 | 제작 진행 중 | 파랑 |
| 👀 검토중 | 초안 전달, 고객 검토 중 | 보라 |
| 🔄 수정중 | 수정 요청 작업 중 | 주황 |
| ✅ 완료 | 납품 완료 | 초록 |
| ❌ 취소 | 주문 취소 | 빨강 |

## 📝 주문 접수 체크리스트

### 문의 접수 시
- [ ] 고객 정보 기록 (이름, 연락처)
- [ ] 서비스 유형 확인
- [ ] 상세 요구사항 파악
- [ ] 참고 이미지/자료 수집
- [ ] 납품 기한 확인
- [ ] 견적 산정

### 작업 시작 전
- [ ] 결제 확인
- [ ] 주문번호 발급
- [ ] 작업 폴더 생성
- [ ] 납품 예정일 안내

### 작업 완료 후
- [ ] 품질 체크
- [ ] 파일 정리 (납품용/원본)
- [ ] 납품 링크 생성
- [ ] 고객에게 전달
- [ ] 리뷰 요청

## 📁 파일 관리 규칙

### 폴더 구조

```
작업폴더/
├── 2025-11/
│   ├── AS-2025-0001_고객명_제품이미지/
│   │   ├── 01_원본/           # 고객 제공 원본
│   │   ├── 02_작업파일/       # PSD, AI 등
│   │   ├── 03_납품/           # 최종 납품 파일
│   │   └── 참고자료/          # 레퍼런스
│   │
│   ├── AS-2025-0002_고객명_썸네일/
│   └── ...
```

### 파일명 규칙

```
[주문번호]_[고객명]_[서비스유형]_[버전].확장자

예시:
AS-2025-0001_ABC마켓_상세페이지_v1.jpg
AS-2025-0001_ABC마켓_상세페이지_v2_수정.jpg
AS-2025-0001_ABC마켓_상세페이지_최종.jpg
```

## 💰 정산 관리

### 월별 정산표

| 항목 | 내용 |
|------|------|
| 총 매출 | 주문 금액 합계 |
| 플랫폼 수수료 | 크몽/숨고 수수료 (약 15%) |
| AI 도구 비용 | Midjourney, Runway 등 |
| 순수익 | 총 매출 - 비용 |

### 수수료 계산

```
크몽: 주문금액 × 20% (VAT 별도)
숨고: 건당 연결료 약 3,000~10,000원
직접 거래: 수수료 없음 (결제 대행 수수료만)
```

## 📞 고객 응대 템플릿

```

문의해 주신 [서비스명] 견적 안내드립니다.

📦 서비스: [서비스 유형]

📅 납품 예정: 결제 후 [N]일 이내
✅ 포함 사항:
   - [항목 1]
   - [항목 2]
   - 수정 [N]회 포함

```

```

📎 파일 확인: [링크]

확인 후 수정사항 있으시면 말씀해 주세요.
수정은 [N]회까지 무료로 진행됩니다.

감사합니다!
```


## 📋 Notion 템플릿 설정 가이드

### 1. 새 데이터베이스 생성
1. Notion에서 새 페이지 생성
2. `/database` 입력 → "Table - Full page" 선택

### 2. 필드 추가
위의 "필수 필드" 섹션 참고하여 속성 추가

### 3. 뷰 생성
- **칸반 보드**: 상태별 관리
- **캘린더**: 납품일 관리
- **테이블**: 전체 목록

### 4. 필터/정렬 설정
- 진행중인 주문만 보기
- 납품일 순 정렬
- 이번 달 주문만 보기
